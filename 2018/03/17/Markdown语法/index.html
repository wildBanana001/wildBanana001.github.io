2018-03-17 PV:

Markdown语法

初次接触markdown是写文章时所需。这是一种轻量级标记语言，创始人为John Gruber(约翰·格鲁伯)。这是一种比HTML还简单容易学习的语言！简单来说就是一种排版语法。

Markdown特性

markdown允许人们使用易读易写的纯文本格式编写文档，然后转化为有效的HTML文档。它吸收了很多在电子邮件中已有的纯文本标记的特性。

分级标题

1. # 一级标题
2. ## 二级标题
3. ### 三级标题
4. #### 四级标题
5. ##### 五级标题
6. ###### 六级标题
7. 一级标题
==================
8. 二级标题
-------------------

斜体和斜体及删除线

1. *斜体*
2. _斜体_
3. **粗体**
4. ***加粗斜体***
5. ~~删除线~~

超链接

1. 链接
[链接标题](链接地址) | [title](url)
例 ： [blog](wildBanana001.github.io)

2. 引用链接
[链接文字][链接标记] | [链接文字][]
例 ： 今天我去了[GitHub][1]社区，创建了一个[账号][2]。这真的是个有趣的[博客][]呢！

[1]:https://www.github.com "GitHub"
[2]:https://www.github.com/wildBanana001 "账号"
[博客]:wildBanana001.github.io

3. 自动链接
<wildBanana001.github.io>

4. 页内超链接
用于跳转到指定标题。
### 超链接{#jump}

我想回到[超链接](#jump)

列表

1. - 无序列表
2. 1. 有序列表

引用

1. > 一级
2. >> 二级
3. >> 三级

插入图片

![图片Alt](图片地址 "图片Title")
例 ： ![我的图片](img.jpg "我的图片")

表格

1.  学号|姓名|分数
    -|-|-
    小明|男|75
    小红|女|79
    小陆|男|92
2.  |学号|姓名|分数|
    |-|-|-|
    |小明|男|75|
    |小红|女|79|
    |小陆|男|92|

分隔线

1. * * *
2. ***
3. *****
4. - - -
5. ---------------------------------------

插入代码

单行在代码前后加入" ` "
例 ： ` markdown `
多行则在代码前后加入" 

       markdown
       markdown
       markdown
