顺便说说四大主流浏览器内核

浏览器内核：负责对网页语法的解释（如HTML,JAVASCRIPT)并渲染网页。

  1. Trident: IE (微软开发，开放内核)
  2. Gecko: FireFox （完全开源）
  3. Presto: Opera（前内核，现已废弃）
  4. Webkit: chrome Safari 傲游3 猎豹浏览器 百度浏览器 opear浏览器(Safari内核,Chrome内核原型,开源)
  • Trident内核速度是四大主流浏览器内核中最慢的。所以曾经国内厂商开发了部分浏览器有”双核”或”多核”版本，就是将Trident内核称为”兼容模式”，其他内核称为”高速模式”。
  • Presto内核在2003年的Opera7中首次被使用，该款引擎的特点就是渲染速度的优化达到了极致，然而代价是牺牲了网页的兼容性。
  • Webkit是苹果公司自己的内核，google的chrome也使用webkit作为内核。 是一个开源的浏览器引擎，同时WebKit 也是苹果Mac OS X 系统引擎框架版本的名称，它拥有清晰的源码结构、极快的渲染速度，包含的 WebCore 排版引擎和JavaScriptCore解析引擎，均是从KDE的KHTML及KJS引擎衍生而来。
  • chromium是 Google 的chrome浏览器背后的引擎，Chromium是一个由Google主导开发的网页浏览器，其目的是为了创建一个安全、稳定和快速的通用浏览器。Chromium相当于Chrome的工程版或称实验版
